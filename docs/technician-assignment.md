### FR 3.1 - Technician Assignment
Allow GS Personnel to assign technicians to specific requests

---

#### Use Case: Assign Technician
Personnel assigns a technician to a repair request.

|  | Description |
|--------------|-----------------|
| **Actors** | Personnel |
| **Preconditions** | Request requires repair |
| **Postconditions** | Technician is assigned and notified |

---

#### Basic Flows

| Actor Action | System Response |
|-------------|-----------------|
| 1. Personnel selects request. | 1.1 Displays list of available technicians. |
| 2. Personnel assigns technician. | 2.1 Saves assignment. |
