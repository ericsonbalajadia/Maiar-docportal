## _Track Request – Add Comments_
Requester adds comments to an existing request.


|  | Description |
|--------------|-----------------|
| **Actors** | Requester |
| **Preconditions** | Request exists |
|| Requester is viewing the request |
| **Postconditions** |  Comment is saved with the request|

#### Basic Flows

| Actor Action | System Response |
|-------------|-----------------|
| 1. Requester enters a comment. | 1.1 System displays the comment. |
| 2. Requester clicks Post Comment. | 2.1 System saves the comment. |
