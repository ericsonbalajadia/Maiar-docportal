### Update Request Status – Add Comments
Personnel/Clerk adds comment when updating status.

|  | Description |
|--------------|-----------------|
| **Actors** | Personnel, Clerk |
| **Preconditions** | Request is open for update |
| **Postconditions** | Comment saved with status update |

## Basic Flows

| Actor Action | System Response |
|-------------|-----------------|
| 1. Personnel/Clerk enters comment. | 1.1 System displays comment. |
| 2. Personnel/Clerk submits comment. | 2.1 System saves comment. |
