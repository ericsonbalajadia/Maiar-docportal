### FR 3.0 - Repair/Service Scheduling
Allow GS Personnel to schedule actual repair/maintenance activities

---

### Use Case: Schedule Repair
Allows personnel to schedule repair tasks for requests.

|  | Description |
|--------|-------------|
| **Actor** | Personnel |
| **Precondition** | Request is approved and requires repair |
| **Postcondition** | Repair schedule is recorded in the system |

#### Basic Flows

| Actor Action | System Response |
|-------------|-----------------|
| 1. Personnel selects request to schedule. | 1.1 Displays available time slots and technicians. |
| 2. Personnel schedules repair. | 2.1 Saves schedule and notifies technician and requester. |
